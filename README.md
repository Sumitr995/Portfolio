## 🚀 Getting Started

To run this portfolio locally:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Sumitr995/Portfolio.git
   ```
2. **Install dependencies:**
   ```bash
   bun install
   # or
   npm install
   ```
3. **Start the development server:**
   ```bash
   bun dev
   # or
   npm run dev
   ```
